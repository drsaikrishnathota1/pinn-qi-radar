# PINN-QI Radar

**PINN-QI Radar: Physics-Informed AI Optimization of Quantum Illumination Detection Bounds in Hypersonic Plasma-Sheath Channels**

This repository contains a simulation-level research codebase for a short communication paper on physics-informed AI optimization of quantum illumination radar bounds under modeled hypersonic plasma-sheath channel degradation.

## Core Scope

- Physics-informed neural surrogate for plasma-sheath channel behavior
- Quantum illumination detection-bound computation
- AI-based optimization of QI operating parameters
- Baseline comparison against classical, fixed-QI, random-search, and grid-search methods
- Simulation-level validation only, not real hardware radar validation
